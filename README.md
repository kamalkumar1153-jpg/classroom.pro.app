# 🌟 Live Classroom - Kamal Sir

एक सरल और पावरफुल **ऑनलाइन लाइव क्लासरूम** वेबसाइट जहाँ छात्र आसानी से पढ़ाई कर सकते हैं।

Firebase से लाइव वीडियो अपडेट होता है, AI Teacher सवालों के जवाब देता है, और प्रोग्रेस ट्रैकर छात्रों को मोटिवेट रखता है।

### ✨ Features

- **Live YouTube Video** — Sir जब चाहें, वीडियो तुरंत बदल सकते हैं (Firebase से)
- **AI Teacher** — Physics & Chemistry के बेसिक सवालों के जवाब (Force, Ohm's Law, Acids आदि)
- **Daily Progress Tracker** — आज कितने घंटे पढ़ाई की, वो ट्रैक करें (12 घंटे तक)
- **Responsive Design** — मोबाइल और डेस्कटॉप दोनों पर अच्छा काम करता है
- **Real-time Updates** — वीडियो बदलते ही सभी छात्रों को नया वीडियो दिखने लगता है

### 🛠️ Technologies Used

- HTML5, CSS3, JavaScript
- Firebase Realtime Database (Live Video Control)
- YouTube Embed API
- LocalStorage (Progress Save करने के लिए)
